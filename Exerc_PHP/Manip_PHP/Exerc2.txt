Exercícios
Exercício 1: 
- Crie um script que gere um arquivo.txt, abra-o e guardar 3 frases nele.
- Crie também um script que imprima o conteúdo gravado no arquivo.

Exercício 1.1: 
Crie um programa que armazene em um arquivo TXT o exemplo da tabuada da aula passada, recebendo no HTML o número a ser calculado
• No script PHP, crie uma função que receba o número e calcule a tabuada dele(ex. 10: 1*10 = 10, 2*10 = 20...).
• A função deve guardo o resultado da tabuada calculada num arquivo TXT. O script deverá ser capaz de imprimir os valores salvos no arquivo.

Exercício 2: 
Aproveitando a estrutura de arquivo JSON criada anteriormente, você deve Modificar o projeto para que, após o cadastro de um novo usuário, a
página exiba a lista completa de todos os usuários já salvos no arquivo JSON.
• Para isso você deve criar uma página com script PHP que leia o arquivo e liste todos os usuários cadastrados.
• Utilize as funções:
– file_get_contents: (obter conteúdo do arquivo),
– json_decode: (converter de JSON para array PHP),
– header(‘Location: lista_usuarios.php’): redirecionar para a página que lista os usuários cadastrados.
